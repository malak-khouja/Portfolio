Movie Rental Duration Prediction

This project predicts the number of days a movie is rented using machine learning models. The workflow includes data preparation, feature engineering from rental metadata, exploratory data analysis, and model training with hyperparameter optimization.

Key Features:

  • Data Loading & Cleaning: Parsed dates, handled types, and verified integrity.
  • Feature Engineering: Created `rental_length_days` and binary flags from special features.
  • Exploratory Analysis: Distribution plots, correlation heatmap, and amount vs. duration trends.
  • Modeling: Compared Decision Tree and Random Forest regressors using GridSearchCV.
  • Evaluation: Reported CV, train, and test MSE; selected the best model.

Technologies Used:

  • Language: Python
  • Libraries: pandas, numpy, scikit-learn, seaborn, matplotlib
  • Tools: Jupyter Notebook

Outcomes: A documented pipeline for predicting rental durations and insights into factors influencing rental periods.

Project information

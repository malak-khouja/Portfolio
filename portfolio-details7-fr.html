Chatbot pour FAQ

Ce projet de chatbot a été développé lors d'un stage CodeAlpha. Il exploite des techniques de NLP pour faire correspondre les questions des utilisateurs à un ensemble de données de questions fréquemment posées (FAQ) et retourner la réponse la plus pertinente. Il est conçu pour aider les utilisateurs, particulièrement dans le domaine de la santé mentale, en fournissant des réponses immédiates et pertinentes.

Fonctionnalités clés :

  • Prétraitement avec NLTK (conversion en minuscules, suppression de la ponctuation et des caractères spéciaux, tokenisation, suppression des mots vides, lemmatisation)
  • Vectorisation TF-IDF pour l'extraction de caractéristiques
  • Similarité cosinus pour faire correspondre les questions
  • Interface chatbot simple en ligne de commande
  • Utilise un ensemble de données FAQ sur la santé mentale

Technologies utilisées :

  • Langage : Python
  • Bibliothèques : Pandas, NLTK, Scikit-learn
  • Similarité : TF-IDF + Similarité cosinus
  • Tokenizer : TensorFlow Keras Tokenizer

Comment ça fonctionne :

  • Les données FAQ sont prétraitées à l'aide de techniques NLP standard.
  • Les vecteurs TF-IDF sont calculés pour toutes les questions.
  • L'entrée de l'utilisateur est comparée aux questions vectorisées à l'aide de la similarité cosinus.
  • Le chatbot renvoie la réponse correspondant à la question la plus similaire.

Impact du projet : Ce projet peut être adapté pour une utilisation dans le support client, les FAQ éducatives ou les ressources de santé mentale, offrant un moyen simple et efficace de répondre aux requêtes répétitives des utilisateurs avec précision.

Informations sur le projet

  • Catégorie : Python / NLP
  • Démo en direct : Google Colab

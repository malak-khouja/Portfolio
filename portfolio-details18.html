Crop Recommendation from Soil Measures

This project predicts the recommended crop using soil nutrients (N, P, K) and pH. It benchmarks simple classifiers to see which single feature carries the strongest predictive signal, and reports the top-performing feature/model pair.

Key Features:

  • EDA & Data Quality: Load soil_measures.csv, inspect schema, detect missing values.
  • Preprocessing: Train/test split with fixed seed; numeric standardization via StandardScaler.
  • Model Benchmarks: Logistic Regression and KNN evaluated feature-by-feature with weighted F1.
  • Model Selection: Prints the best model per feature and the top single feature overall.

Tech Stack:

  • Python, pandas, scikit-learn
  • Jupyter Notebook for the full workflow

How to Run:

  • Install deps: pip install pandas scikit-learn jupyter
  • Open notebook.ipynb in Jupyter and run cells in order.

Outcome: The notebook prints the highest F1-score per feature and highlights the best single predictor to guide planting decisions.

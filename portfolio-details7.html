Chatbot for FAQ

This chatbot project was developed during a CodeAlpha internship. It leverages NLP techniques to match user questions to a curated dataset of frequently asked questions (FAQs) and return the most relevant response. It’s designed to assist users—particularly in the mental health domain—by providing immediate and relevant answers.

Key Features:

  • Preprocessing with NLTK (lowercasing, punctuation & special character removal, tokenization, stopword removal, lemmatization)
  • TF-IDF vectorization for feature extraction
  • Cosine similarity for matching questions
  • Simple command-line chatbot interface
  • Uses a mental health FAQs dataset

Technologies Used:

  • Language: Python
  • Libraries: Pandas, NLTK, Scikit-learn
  • Similarity: TF-IDF + Cosine Similarity
  • Tokenizer: TensorFlow Keras Tokenizer

How It Works:

  • FAQ data is preprocessed using standard NLP techniques.
  • TF-IDF vectors are computed for all questions.
  • User input is compared to the vectorized questions using cosine similarity.
  • The chatbot returns the answer corresponding to the most similar question.

Project Impact: This project can be adapted for use in customer support, educational FAQs, or mental health resources, offering a simple and effective way to answer repetitive user queries with accuracy.

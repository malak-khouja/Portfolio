Language Translation Tool

The Language Translation Tool is a React-based web application built as part of a CodeAlpha internship. It allows users to translate text between multiple languages using a local language model powered by Ollama. The tool features a clean and responsive UI, real-time translation, and intuitive dropdowns for selecting source and target languages.

Key Features:

  • Language Selection: Choose source and target languages using dynamic dropdowns populated from LibreTranslate.
  • Live Translation: Translate text in real-time using a locally running translation model.
  • Local AI Model: Utilizes Ollama with a local model (Gemma) for secure and offline translation.
  • Clean UI: Responsive interface built with React and Bootstrap for a user-friendly experience.
  • Error Handling: Graceful error messages for failed translations or invalid inputs.

Technologies Used:

  • Frontend: React with Bootstrap (React-Bootstrap)
  • API: LibreTranslate for available language list
  • Local Model: Ollama running Gemma for actual translation processing
  • Communication: Fetch API to communicate with the local translation endpoint

How It Works: The app fetches the available languages from LibreTranslate. Users enter text and select languages, which are then sent to a local translation API. The translated result is streamed back and rendered in real time.

Project Impact: This tool provides an efficient, offline-capable translation system that respects data privacy by avoiding cloud-based translation APIs. It's ideal for local environments, research, or internal tools where internet-based translation isn't viable.

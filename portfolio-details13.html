Passenger Satisfaction Prediction App

The Passenger Satisfaction Prediction App is a full-stack application that predicts airline passenger satisfaction using machine learning and provides an interactive dashboard to visualize metrics and passenger profiles at risk. The project covers all stages from data preprocessing to deployment with a professional frontend and backend.

Project Workflow & Features:

  • Data Preprocessing: Handling missing values, encoding categorical variables, scaling numeric features, and feature selection.
  • Exploratory Data Analysis (EDA): Visualization of distributions, correlations, and satisfaction patterns using Plotly and Matplotlib.
  • Machine Learning: Training a KNN classifier to predict passenger satisfaction, hyperparameter tuning, and evaluation with accuracy and RMSE.
  • Backend: FastAPI application with endpoints for adding passengers, retrieving predictions, and storing data in MongoDB.
  • Frontend: React application with interactive forms, dynamic dashboard, and Plotly.js charts to display metrics by age, class, and service type.
  • Dashboard Analytics: Visualization of satisfied vs dissatisfied passengers, identification of high-risk profiles, and summary statistics for decision-making.

Technologies Used:

  • Frontend: React.js, Plotly.js
  • Backend: Python, FastAPI, Scikit-learn
  • Database: MongoDB
  • Visualization: Plotly, Matplotlib, Seaborn
  • Other Tools: Git, npm, Postman

Impact & Outcome:
This project demonstrates how to integrate machine learning with a full-stack application, providing real-time predictions and actionable insights. Airlines can identify at-risk passengers, improve services, and optimize customer satisfaction.

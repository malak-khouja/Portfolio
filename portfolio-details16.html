Facial Recognition with Supervised Learning

This project classifies faces as Arnie vs Non‑Arnie using supervised learning. The workflow includes data loading, stratified train/test splitting, model selection via cross‑validation, and evaluation with standard metrics and visualizations.

Key Features:

  • Data Loading & Split: Read tabular features from data/lfw_arnie_nonarnie.csv, separate predictors/labels, and use stratified split to preserve class balance.
  • KNN Neighbor Sweep: Evaluate F1 across k values and visualize the bias‑variance trade‑off to pick an optimal neighborhood size.
  • Model Selection: Compare KNN, Logistic Regression, and SVM with scaling inside pipelines; tune hyperparameters via GridSearchCV with KFold.
  • Evaluation: Report Accuracy, Precision, Recall, and F1; inspect the confusion matrix to understand per‑class performance.

Technologies Used:

  • Language: Python
  • Libraries: pandas, numpy, scikit‑learn, matplotlib, seaborn
  • CV & Metrics: KFold, GridSearchCV, confusion_matrix, Accuracy/Precision/Recall/F1

Project Impact: A clear, reproducible pipeline for facial recognition that demonstrates model comparison, tuning, and interpretation with visuals.

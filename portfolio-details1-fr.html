Système de gestion des utilisateurs

Le Système de gestion des utilisateurs est une application web full-stack CRUD qui permet aux administrateurs de gérer les utilisateurs de manière efficace. Le projet présente une interface utilisateur moderne et responsive alimentée par React, Redux et React-Bootstrap, avec un backend robuste construit à l'aide de Node.js, Express et MongoDB.

Fonctionnalités clés :

  • Créer : Ajouter de nouveaux utilisateurs avec des détails tels que l'e-mail, le prénom, le nom et l'âge.
  • Lire : Afficher tous les utilisateurs dans un tableau responsive stylisé avec Bootstrap.
  • Mettre à jour : Modifier les informations utilisateur de manière transparente (UI + backend intégré).
  • Supprimer : Supprimer des utilisateurs instantanément (UI + backend intégré).
  • Actualisation automatique : Le tableau se met à jour dynamiquement après les opérations CRUD.
  • Design responsive : Mise en page moderne et mobile-first utilisant React-Bootstrap.
  • Gestion d'état : Redux entièrement intégré pour une gestion d'état cohérente et évolutive.

Technologies utilisées :

  • Frontend : React, Redux, React-Bootstrap, Axios
  • Backend : Node.js, Express.js, MongoDB, Mongoose
  • Autres outils : Git, npm

Impact du projet : Ce système démontre comment intégrer un workflow full-stack MERN (MongoDB, Express, React, Node.js), ce qui en fait une base idéale pour des applications d'entreprise évolutives. Il fournit une interface intuitive pour gérer les données tout en présentant une solide intégration backend.

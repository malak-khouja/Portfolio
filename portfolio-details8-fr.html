Outil de traduction linguistique

L'outil de traduction linguistique est une application web basée sur React construite dans le cadre d'un stage CodeAlpha. Il permet aux utilisateurs de traduire du texte entre plusieurs langues en utilisant un modèle de langage local alimenté par Ollama. L'outil présente une interface utilisateur propre et responsive, une traduction en temps réel et des menus déroulants intuitifs pour sélectionner les langues source et cible.

Fonctionnalités clés :

  • Sélection de langue : Choisissez les langues source et cible à l'aide de menus déroulants dynamiques remplis depuis LibreTranslate.
  • Traduction en direct : Traduisez du texte en temps réel à l'aide d'un modèle de traduction local en cours d'exécution.
  • Modèle d'IA local : Utilise Ollama avec un modèle local (Gemma) pour une traduction sécurisée et hors ligne.
  • Interface utilisateur propre : Interface responsive construite avec React et Bootstrap pour une expérience conviviale.
  • Gestion des erreurs : Messages d'erreur gracieux pour les traductions échouées ou les entrées invalides.

Technologies utilisées :

  • Frontend : React avec Bootstrap (React-Bootstrap)
  • API : LibreTranslate pour la liste des langues disponibles
  • Modèle local : Ollama exécutant Gemma pour le traitement de traduction réel
  • Communication : API Fetch pour communiquer avec le point de terminaison de traduction local

Comment ça fonctionne : L'application récupère les langues disponibles depuis LibreTranslate. Les utilisateurs saisissent du texte et sélectionnent des langues, qui sont ensuite envoyées à une API de traduction locale. Le résultat traduit est diffusé en retour et affiché en temps réel.

Impact du projet : Cet outil fournit un système de traduction efficace et capable de fonctionner hors ligne qui respecte la confidentialité des données en évitant les API de traduction basées sur le cloud. Il est idéal pour les environnements locaux, la recherche ou les outils internes où la traduction basée sur Internet n'est pas viable.
